# ערב 10 – חבילה נקייה מוכנה להחלפה

קבצים מרכזיים:
- signup.html – הרשמה חדשה עם טלפון + קוד אישי + אימות קוד.
- client-v3.html – תיקוני שמירה, כפתור עזרה, מידע על מעל 10 לחמניות ותצוגה נקייה יותר.
- manager.html – מחיר 22 ₪, ספירת נמסרו/פתוחים וסנכרון סטטוסים מהשליח.
- courier.html – תיעוד צילום עם שעה ותאריך, בלי שליחת וואטסאפ ללקוח בזמן החלוקה.
- bakery.html – התקנה, טיימר, צפצוף ותזכורת.
- sw.js – cache v11.
- supabase-v2-upgrade.sql – טבלאות עזר לגרסאות, התראות, לוגים ותיעוד.

החיבור ל-Supabase נשאר דרך supabase-config.js הקיים.

העלאה מומלצת:
1. לשמור גיבוי של האתר הישן.
2. להעלות את הקבצים האלה במקום הקיימים.
3. להריץ את supabase-v2-upgrade.sql אם עוד לא רצת.
4. לפתוח את signup.html ואז client-v3.html ולבדוק הרשמה/כניסה.
